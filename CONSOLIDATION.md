# Repository consolidation

Completed on 2026-07-13.

## Canonical repository

`alifazelidehkordi/chatgpt-mindmap-to-xmind` is the single active repository
for the ChatGPT-to-XMind workflow. It contains the Playwright/Patchright
automation, parallel workers, OPML repair and validation, and the reliability
improvements from earlier implementations.

## Archived repositories

- `chatgpt-mindmap-to-xmind-legacy` — original Selenium implementation
- `chatgpt-mindmap-to-xmind-v2` — duplicate/reverted iteration
- `chatgpt-mindmap-pipeline` — reliability-focused predecessor
- `chatgpt-mindmap-automation` — OPML-only prototype

Their GitHub URLs retain redirects where GitHub supports them, and each
repository has a migration README pointing here. They are retained for history
and old releases only.
